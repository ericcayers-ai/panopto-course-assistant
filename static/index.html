📘 Import a Moodle course

Connect to your Moodle site and pick a course. The assistant reads it through Moodle's official mobile web-service API — exact titles, file types and lecture feeds, with no page-scraping or guesswork — then prepares everything for export.

1

Connect to Moodle

Enter your Moodle site (or any course link on it), for example https://elearn.waikato.ac.nz.

Used once, on this computer, to obtain a web-service token from your Moodle site. Your password is never stored.

  1. Click Open Moodle security keys ↗ below — it opens your Moodle security keys page in a new tab.
  2. Find the Moodle Mobile Web Service section and copy that token. (Not the RSS token — it won't work.)
  3. If that section is missing, open the Moodle mobile app, sign in via your institution, then return to step 2.

The assistant will import the outline, documents, lectures and images for the selected course. For finer control, choose Advanced from Change workspace.
