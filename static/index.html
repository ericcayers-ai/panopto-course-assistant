Text to speech

Reads a Markdown file aloud using VibeVoice-Realtime-0.5B. The model runs locally and downloads from Hugging Face on first use (about 1 GB). A GPU is strongly recommended; CPU generation is very slow.

VibeVoice is not installed. From the project folder run pip install -r requirements-tts.txt, then restart. The model and voices download on first use.
