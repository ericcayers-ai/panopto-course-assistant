🗃️ Notion study database (CSV)

One row per lecture — Name, Week, Date, Topic, Tags, Status, Summary. In Notion: Import → CSV, then set Tags to Multi-select. With an AI provider configured the Summary column is LLM-written. Runs as a job.

🎬 Lecture transcripts + SRT

Export transcript text and .srt subtitle files for all recorded lectures. Place SRT files in the same folder as your video recordings so media players load them automatically.

🃏 Anki flashcards Anki-ready

Uses your local LLM to generate Q&A, cloze and definition cards from your transcripts, auto-tagged by course · week · topic. Exports an Anki-importable .txt and a .csv to the folder you choose. Requires Ollama (installed with extra dependencies).
