Live mic:

Reads a Markdown file aloud using Kokoro-82M (open-weight, Apache-2.0). Built for long-form narration: text is chunked with voice continuity and stitched into one WAV. Runs well on CPU; a GPU speeds things up if available. The model downloads from Hugging Face on first use (~300 MB).

Kokoro TTS is not installed. From the project folder run pip install -r requirements-tts.txt, then restart. The model downloads on first use.

Generate speech from a Markdown file

Voices ship with Kokoro. Long lectures are chunked automatically so quality stays stable.
