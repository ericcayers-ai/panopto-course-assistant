# VibeVoice text-to-speech (Voice TTS tab).
#
#   pip install -r requirements-tts.txt
#
# This installs Microsoft VibeVoice (Realtime-0.5B streaming TTS) and all of its
# dependencies (torch, transformers, diffusers, librosa, …) straight from GitHub
# — the package is not published on PyPI. The [streamingtts] extra pins the
# transformers version the streaming model needs.
#
# The ~1 GB model is downloaded from Hugging Face on first generation, and the
# small per-voice preset files are fetched on demand — no manual repo clone or
# voice download is required.
#
# A CUDA GPU is strongly recommended. For best speed, optionally also install
# flash-attention (`pip install flash-attn --no-build-isolation`); without it
# the app automatically falls back to PyTorch SDPA attention.

vibevoice[streamingtts] @ git+https://github.com/microsoft/VibeVoice.git
